# stylint-config

> Just a custom stylint config

## Installation

[npm](https://www.npmjs.com/package/@schascha/stylint-config)

```
npm install @schascha/stylint-config --save-dev
```

### Usage

Set your `stylelint` config to:

```
{
  "extends": "@schascha/stylint-config"
}
```
