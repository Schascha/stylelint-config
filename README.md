# stylelint-config

> Just a custom stylelint config

## Installation

[npm](https://www.npmjs.com/package/@schascha/stylelint-config)

```
npm install @schascha/stylelint-config --save-dev
```

### Usage

Set your `stylelint` config to:

```
{
  "extends": "@schascha/stylelint-config"
}
```
